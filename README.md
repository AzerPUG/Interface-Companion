# Description and Features
AzerPUG's Interface Companion adds a companion to you User Interface (location customizable) to keep you company on your quests and travels.
'Do you want to be my friend? I want to be always with you!'

# Bugs, Errors, Glitches and Requests.
If you find any of the above, please do not hesitate to create a GitHub Issue here: https://github.com/AzerPUG/Interface-Companion/issues  
Or you can join our Discord and let us know in the AzerTech channel: http://www.azerpug.com/discord  
Please note that our Discord Verification level prevents new people from talking/commenting/ranking in the first 10 minutes!
